# Dra. Fabi Cristine Veterinária

Site profissional e responsivo construído com React, Vite e TailwindCSS para apresentação de atendimento veterinário domiciliar no Rio de Janeiro.

## Tecnologias

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Como usar

1. Instale as dependências:

```bash
npm install
```

2. Inicie o projeto em modo de desenvolvimento:

```bash
npm run dev
```

3. Gere o build de produção:

```bash
npm run build
```

4. Pré-visualize o build:

```bash
npm run preview
```

## Estrutura principal

- `src/App.tsx` — composição das seções da página
- `src/components/` — componentes reutilizáveis
- `src/index.css` — estilos globais e configuração Tailwind
- `public/` — recursos estáticos como imagens e ícone

## Recursos implementados

- Menu fixo com scroll suave
- Seção hero com CTA
- Sobre, serviços, diferenciais, depoimentos, galeria e contato
- Formulário de contato funcional via `mailto:`
- Botão flutuante de WhatsApp
- Animações suaves com Framer Motion
- Layout mobile responsivo
- SEO básico com meta tags e Open Graph
